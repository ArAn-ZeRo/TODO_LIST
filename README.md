# Todo App

A simple and responsive Todo application built with React and Tailwind CSS.

## Features

* Add new tasks
* Delete tasks
* Responsive UI
* Built with React Hooks
* Styled using Tailwind CSS

## Technologies Used

* React
* JavaScript (ES6+)
* Tailwind CSS
* Vite

## Installation

1. Clone the repository:

```bash
git clone https://github.com/ArAn-ZeRo/TODO_LIST.git
```

2. Navigate to the project directory:

```bash
cd TODO_LIST
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

## How It Works

* Users can enter a task in the input field.
* Clicking the **Add** button adds the task to the list.
* Each task includes a **Delete** button.
* Clicking **Delete** removes the selected task from the list.
